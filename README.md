# Supply Chain Management DApp (Polygon Amoy)

A decentralized Supply Chain Management system developed using **Solidity**, **Hardhat**, and **React.js**, deployed on the **Polygon Amoy Testnet**.  
The DApp ensures **transparency, traceability, and immutability** of product movement from **Manufacturer → Distributor → Retailer → Customer**.

## Project Objective
The objective of this project is to design and implement a blockchain-based supply chain solution that:

- Records product creation by manufacturers
- Tracks ownership transfers across supply chain roles
- Maintains an immutable audit trail
- Prevents tampering and ensures trust among stakeholders

---

## System Roles
The system supports the following roles:

- **Manufacturer** – Registers new products
- **Distributor** – Receives and transfers products
- **Retailer** – Receives products from distributors
- **Customer** – Final product owner

Each role interacts with the smart contract through blockchain transactions.
